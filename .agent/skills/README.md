# ProdeX Skills Index

This directory contains the AI agent's specialized knowledge and instructions, restructured by project stack.

## 📂 Structure

### [core/](./core)
The essential rules and identity of the ProdeX dropshipping marketplace.
- `project-identity`: The source of truth for ProdeX development.
- `using-superpowers`: How the agent finds and uses skills.
- `codebase-context`: Master repository map, settings helpers, user roles, and helper structures.

### [laravel/](./laravel)
Official Laravel agent skills for code refinement, cloud deployment, and observability.
- `laravel-simplifier`: Expert PHP/Laravel code simplification.
- `deploying-laravel-cloud`: Managing Laravel Cloud infrastructure.
- `configure-nightwatch`: Configuring Nightwatch data collection.
- `custom-localization-and-translations`: String wrapping rules using the `translate()` helper.
- `aiz-uploader-integration`: Custom file picker data attributes and asset resolution.
- `payment-gateway-integration`: Unified callback workflows for cart and package checkouts.

### [engineering/](./engineering)
Backend logic, reasoning capabilities, and quality assurance.
- **Reasoning**: `writing-plans`, `executing-plans`, `brainstorming`.
- **Quality**: `test-driven-development`, `systematic-debugging`, `webapp-testing`.
- **Backend**: Specific PHP or Laravel logic patterns.

### [ui-ux/](./ui-ux)
Premium design, aesthetics, and visual guidelines.
- `ui-ux-pro-max`, `frontend-design`, `theme-factory`, `web-design-guidelines`, `web-artifacts-builder`.
